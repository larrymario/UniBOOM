操作：
鼠标移动：移动视角
鼠标滚轮：拉近/远视角
鼠标左键：隐藏/显示鼠标
WASD：移动人物
Space：放置炸弹
E：隐藏/显示鼠标
ESC：退出
R：重新加载

现在稍微有点炸弹人的样子了，不过还是只能炸炸方块，红色的小球球并不是敌人，她们不会伤害你，你也伤害不了她们。
火力固定为5，炸弹数固定为30。

已知BUG或缺陷：
* 在同一个地方可以重复放置多个炸弹。
* Unitychan可能会突然飘起来。
* 鼠标好像隐藏不掉。
* 场景里的各种东西会挡住视线。
* 可能会有各种谜之卡顿。
